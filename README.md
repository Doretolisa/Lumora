# ⚡ Lumora

![Logo do projeto](./logo.png)

## 📌 Sobre o projeto

O **Lumora** é um projeto voltado para geração alternativa de energia, composto por dois tipos de geradores:

* 🔋 Gerador a base de resíduos orgânicos
* 💧 Gerador a base de água

A proposta é oferecer uma solução acessível para uso doméstico, especialmente em regiões com infraestrutura limitada ou em situações de emergência, como quedas de energia causadas por desastres naturais.

---

## 🎯 Objetivo

O projeto busca reduzir os impactos da falta de energia em bairros desfavorecidos, proporcionando uma alternativa sustentável, acessível e eficiente para o usuário comum.

---

## 🌐 Plataforma

O sistema conta com:

* Um **site institucional**, que apresenta a ideia e explica o funcionamento dos geradores
* Um **sistema com login**, onde o usuário pode:

  * Gerenciar o consumo de energia
  * Acompanhar dados do gerador
  * Utilizar futuras funcionalidades em desenvolvimento

Além disso, o projeto prevê um **aplicativo mobile**, focado no gerenciamento do sistema.
